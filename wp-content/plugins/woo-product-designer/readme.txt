== Description ==

Personalized Product Option Manager Plugin is the best plugin to allow users to personalize a product by providing additional information while ordering. Clients can select among options like (Size, Color, etc), choose images and even upload Images/Files before ordering with WooCommerce. Sometime clients need more options/controls to Personalize their order specially when itâ€™s related to Prints, Designs and Arts etc.

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. After activation, You will option page `Product Designer`
