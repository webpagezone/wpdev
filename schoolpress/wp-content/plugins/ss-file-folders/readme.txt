== Description ==

Adds a [file-folder] shortcode which will show links to the parent page (../), any sub page, and any attached file.

== Installation ==

1. Upload the `ss-file-folders` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Add the shortcode [file-folder] to any page/post that you want file folder functionality. 
1. Add files via the media uploader.
